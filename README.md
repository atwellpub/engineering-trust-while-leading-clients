# Engineering Trust & Leading Clients

**[View the presentation](https://atwellpub.github.io/engineering-trust-while-leading-clients/presentation/)**

An interactive HTML slide presentation built for a Codeable Skillset Chat. The presentation covers practical strategies for building client trust, improving communication, and growing professional relationships on the Codeable platform.

## Structure

```
presentation/       # Main slide deck
  index.html        # 42 slides across 3 arcs + recap
  styles.css        # Warm navy theme with step-reveal animations
  script.js         # Slide navigation, step animations, keyboard controls
assets/             # Images, SVGs, and GIFs used in slides
outline/            # Presentation outline and planning docs
```

## Arcs

- **Arc I - Improved Acquisition Strategies**: The Rhetorical Triangle (Ethos, Pathos, Logos), Mirroring & Expectation Management
- **Arc II - Improved Retention Strategies**: The Sandwich Method (Opening PDF, Work Execution, Closing PDF), presentation and value delivery
- **Arc III - Future Preparation & Opportunity Management**: The Stone Sorter parable, Always Be Opening mentality

## Running

Open `presentation/index.html` in a browser. No build step required.

### Controls

| Key | Action |
|-----|--------|
| Right Arrow / Space | Next step or slide |
| Left Arrow / Backspace | Previous slide |
| Home | First slide |
| End | Last slide |

Click anywhere on the slide to advance as well.

## Features

- Step-by-step reveal animations for bullets, quotes, images, and cards
- Card-stack image reveals tied to bullet points
- Half-screen background images with content overlays
- Media reveal slides with headline replacement animations
- Arc-based color theming and progress indicators
- Responsive layout with mobile support
